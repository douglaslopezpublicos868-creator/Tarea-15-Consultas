# Sistema de Gestión de Alojamientos Turísticos

## Descripción

Proyecto SQL desarrollado en PostgreSQL para administrar alojamientos turísticos, propietarios, huéspedes, reservas, pagos y reseñas.

## Motor de Base de Datos

* PostgreSQL

## Contenido del Repositorio

* `db_gestion_alojamientos_turisticos.sql`

  * Script de creación de base de datos
  * Creación de tablas
  * Inserción de datos de prueba

* `consultas_sql.sql`

  * 20 consultas SQL:

    * INSERT
    * SELECT
    * UPDATE
    * DELETE
    * JOIN
    * LEFT JOIN
    * Funciones de agregación
    * HAVING
    * Subconsultas

## Esquema de Base de Datos

Tablas utilizadas:

* propietarios
* alojamientos
* huespedes
* reservas
* pagos
* reseñas
